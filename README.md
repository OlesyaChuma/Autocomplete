# Автодополнение текста на основе префиксного дерева (Trie)

Данный проект реализует **консольное приложение автодополнения текста**  
на основе **префиксного дерева (Trie)**.

Программа позволяет пользователю вводить начало слова  
и предлагает возможные варианты продолжения  
по заранее заданному словарю.

Проект выполнен на языке **C++** в рамках темы  
*Сбалансированные деревья поиска и префиксные деревья*.

---

## Возможности программы

- Автодополнение слов по введённому префиксу.
- Использование структуры данных **Trie (префиксное дерево)**.
- Быстрый поиск по префиксу с временной сложностью **O(n)**.
- Загрузка словаря из текстового файла `dictionary.txt`.
- Использование встроенного словаря при отсутствии файла.
- Проверка корректности ввода:
  - запрет пустой строки;
  - запрет цифр и неалфавитных символов.
- Консольное приложение с циклическим вводом данных.

---

## Используемая структура данных

В основе программы лежит **префиксное дерево (Trie)**:

- каждый узел соответствует одному символу;
- слово представляется путём от корня к конечному узлу;
- автодополнение реализуется обходом поддерева,
  соответствующего введённому префиксу.

Данная структура данных оптимально подходит для задач
автодополнения и поиска по префиксу.

## Как запустить программу

1. Открыть решение в **Visual Studio**.
2. Собрать проект в режиме **Debug** или **Release**.
3. Убедиться, что файл `dictionary.txt` находится
   в каталоге с исполняемым файлом.
4. Запустить программу.
5. Ввести начало слова для получения подсказок.
6. Для завершения работы ввести `exit`.

---

## Формат словаря

Файл `dictionary.txt` должен содержать:

- одно слово в строке;
- только буквенные символы;
- рекомендуется использовать нижний регистр.

Пример содержимого файла:
apple
application
apply
banana
bank
basket

## Примечания

- Все операции автодополнения выполняются в оперативной памяти.
- Проект легко расширяется:
  - добавление учёта частоты слов;
  - регистронезависимый поиск;
  - сохранение новых слов.
